# Baseball / Softball Team Management App

## Platforms
- Tablet
- Web (as proof of concept first?)

## Features
- Create a team, invite people to join
- Manage Stats, log hits, walks, pitches etc
- Create accounts, team owners and players
- Softball, Baseball, Kickball
- Allow users to search teams recruiting in area
- Messaging
- Event organizing, team organizing, allow people to select leagues
